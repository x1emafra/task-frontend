# 🚀 Task App — Full Stack Project

Aplicación full-stack para gestión de tareas con CRUD completo.

## 🌐 Live Demo

👉 https://task-frontend-ten-sigma.vercel.app

## 🧠 Descripción

Task App permite crear, visualizar, actualizar y eliminar tareas en tiempo real.
El proyecto está desplegado en producción utilizando una arquitectura cliente-servidor.

---

## ⚙️ Tech Stack

### Frontend

* React (Vite)
* Axios
* TailwindCSS

### Backend

* Node.js
* Express
* Prisma ORM

### Base de datos

* SQLite / PostgreSQL (según config)

### Deploy

* Frontend: Vercel
* Backend: Render

---

## 🔌 API

Base URL:

https://task-api-emanuel.onrender.com/api

Endpoints:

* GET `/tasks`
* POST `/tasks`
* PUT `/tasks/:id`
* DELETE `/tasks/:id`

---

## ✨ Features

* Crear tareas
* Listar tareas
* Actualizar tareas
* Eliminar tareas
* Persistencia en base de datos
* Deploy en producción
